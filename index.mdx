Comfaca Créditos en Línea moderniza el proceso de solicitud y aprobación de créditos para empleados afiliados a la **Caja de Compensación Familiar del Caquetá (COMFACA)**. La plataforma permite a los afiliados solicitar créditos de forma completamente digital, con carga de documentos, seguimiento en tiempo real y firma electrónica integrada.

## Cómo Empezar

<Steps>
  <Step title="Clonar e instalar dependencias">
    Clona el repositorio e instala las dependencias con pnpm.
    ```bash
    git clone https://github.com/elegroag/nuxt-credito-caja.git
    cd nuxt-credito-caja
    pnpm install
    ```
  </Step>
  <Step title="Configurar variables de entorno">
    Copia la plantilla y rellena las credenciales de base de datos y APIs externas.
    ```bash
    cp .env.template .env
    ```
  </Step>
  <Step title="Inicializar la base de datos">
    Genera el cliente Prisma, ejecuta las migraciones y carga los datos iniciales.
    ```bash
    pnpm db:generate
    pnpm db:migrate
    pnpm db:seed
    ```
  </Step>
  <Step title="Iniciar el servidor de desarrollo">
    Lanza el servidor con hot-reload en el puerto configurado.
    ```bash
    pnpm dev
    ```
  </Step>
</Steps>

<Note>
  Consulta la [guía de configuración completa](/setup/environment) para todos los detalles sobre variables de entorno, integraciones externas y configuración del almacenamiento SFTP.
</Note>
